# Contributing
This repository is a public kit. Contributions are welcome if they improve clarity, safety, or usefulness without adding sensitive details.

Rules
1. Do not add tenant names, tenant ids, internal urls, secrets, tokens, or proprietary data
2. Keep examples illustrative and public safe
3. If you introduce a new rule, add a test case to validate it
4. Update governance/Change_Log.md for any behavior level change

Process
1. Open an issue using the templates
2. Propose changes with impacted files listed
3. Submit a pull request referencing the issue
4. Ensure tests and examples still align with the guardrails
